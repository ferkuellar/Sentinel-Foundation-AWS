# Sentinel-Foundation-AWS

Enterprise-ready AWS secure foundation baseline with Terraform, governance controls, auditability, cost awareness, and CI/CD validation.

## Scope

This project demonstrates:

- multi-account thinking
- governance baseline
- reusable network foundation
- centralized audit and logging patterns
- monitoring and alerting baseline
- cost controls and quota strategy
- Terraform validation pipeline
- architecture decision records
- evidence by phase

## Phases

- Phase 0 — Bootstrap
- Phase 1 — Governance model and account blueprint
- Phase 2 — Network foundation
- Phase 3 — Logging, encryption and audit
- Phase 4 — Governance controls and monitoring
- Phase 5 — Cost controls and quotas
- Phase 6 — CI/CD pipeline
- Phase 7 — Final docs and enterprise roadmap
